Introduction
----------------

Store website model for Python 2 project; handling users login, admin panel, main store, details and cart.

Requirements
----------------

Python 3.10 (Recommended) - Download from https://www.python.org/downloads/
Django 4 (Recommended) - Download from https://www.djangoproject.com/download/ or using pip: "pip install Django==4.0.4"
